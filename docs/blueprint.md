# **App Name**: আমার প্রশ্ন

## Core Features:

- Class & Subject Explorer: Users can navigate through various academic levels (classes 6th to 10th) and select specific subjects to view their curriculum. This establishes the app's foundational content structure as per user's specification for specified books.
- AI Question Generation Tool: An AI-powered tool that generates practice questions tailored to the selected class and subject, aiding in study and self-assessment, and replacing placeholder content.
- PDF Textbook Uploader: Users can upload new PDF textbooks by selecting the target class and subject. The uploaded PDFs are stored and integrated into the relevant curriculum sections.
- Integrated PDF Viewer: Provides an in-app viewer for users to read and interact with all uploaded PDF textbooks seamlessly.
- Persistent Navigation: Implements fixed top and bottom navigation bars, providing consistent access to the main dashboard, settings, and other core sections of the application.

## Style Guidelines:

- Primary Color: A vibrant yet calm blue (#0C66CC), reflecting the clear sky requested, perfect for an educational and focus-oriented application, and providing good contrast on a light background.
- Background Color: A soft, desaturated off-white (#E8F1F6), subtly incorporating the primary blue's hue for a cohesive and light theme, enhancing readability.
- Accent Color: A refreshing teal-green (#2ACCA1), analogous to the primary blue, offering a clear distinction for interactive elements and highlights to guide user attention.
- Headlines and Body Text: 'Inter', a versatile sans-serif font, chosen for its excellent readability and modern, neutral aesthetic, ideal for diverse educational content including questions and book texts.
- Fixed 'bright sky blue' navigation bars at the top and bottom of the screen, ensuring persistent access to primary features and global navigation, as specified by the user's request.
- Use clear and universally recognizable icons for actions like 'Upload', 'Settings', and navigation items like 'Dashboard' and various class/subject categories, maintaining a clean and intuitive user interface.
